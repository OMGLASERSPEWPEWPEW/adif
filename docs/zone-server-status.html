Why Rust over C#

• Memory safety without GC — no garbage collection pauses during combat ticks
• Fearless concurrency — ownership system prevents data races at compile time
• bevy_ecs — composition over EQEmu's Entity→Mob→NPC inheritance (1952-line mob.h)
• Zero-cost abstractions — traits for geometry interfaces without virtual dispatch overhead
• Ecosystem — tokio (async), sqlx (compile-time DB), prost (protobuf) are all production-grade
